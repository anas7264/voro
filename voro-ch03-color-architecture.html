// 02 — Surface System

DEPTH
BY LAYERS

--s3 · Border / Outermost
#251D16
--s2 · Elevated Surface
#1A1410
--s1 · Card Background
#0C0906
--s0 · Page Root
#050403
--brand · Content

Four neutral surfaces create a consistent depth hierarchy. Each layer is ~2× lighter than the one below — providing spatial clarity without relying on color.

  • --s3 — Borders, dividers, outlines
  • --s2 — Hover states, tooltips, popovers
  • --s1 — Cards, panels, modals
  • --s0 — Root page background
  • --brand — Foreground accent on all layers

THE SYSTEM,
IN CODE

/* ══ VORO Color Architecture — 9-Token Adaptive System ══ */ /* Change --brand-h (0–360) to shift the entire brand spectrum. */ :root { /* ── Surface layers (fixed warm-dark) ────────────────────── */ --s0: #050403; /* Deepest bg */ --s1: #0C0906; /* Card backgrounds */ --s2: #1A1410; /* Elevated surfaces */ --s3: #251D16; /* Borders, dividers */ /* ── Adaptive hue (0–360) ────────────────────────────────── */ --brand-h: 38; /* ── Brand spectrum ──────────────────────────────────────── */ --brand-dim: /* #4A2800 */ hsl(var(--brand-h), 70%, 22%); --brand: /* #E89020 */ hsl(var(--brand-h), 85%, 56%); --brand-glow: /* #F5C060 */ hsl(var(--brand-h), 100%, 70%); /* ── Typography ──────────────────────────────────────────── */ --text-primary: rgba(237, 232, 224, 0.95); --text-secondary: rgba(200, 185, 168, 0.65); --text-muted: rgba(140, 124, 108, 0.45); }
