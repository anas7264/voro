// Mark Anatomy

THE
FORGE
MARK

The Voro mark is built from three nested triangles under even-odd fill rule, creating a bold V silhouette with a hidden diamond — the Forge Spark — visible only when you look into the negative space. The strike point at the base represents concentrated force: energy converging to a single point of impact.

// stroke on lit faces · rgba(255,230,150,.45)

// Usage Rules

DO. DON'T.
SIMPLE.

✓ DO — Scale uniformly

Always maintain the original aspect ratio. Scale from center.

✓ DO — Use on approved backgrounds

Black, amber, white, or near-black surfaces only. No mid-tone grays.

✗ DON'T — Rotate the mark

The strike point must always face downward. No rotation, reflection, or skew.

✗ DON'T — Change the mark color

Use only the approved palette variants. No gradients, tints, or off-brand colors.
